What makes an agentic system?
- Autonomy: The ability to operate independently without constant human intervention.
- Goal-Oriented Behavior: The capability to set and pursue specific objectives.
- Adaptability: The ability to learn from experience and adjust behavior accordingly.
- Perception: The capacity to perceive and interpret the environment.
- Decision-Making: The ability to make informed decisions based on available information.
- Communication: The capability to interact with humans or other agents effectively.
- tools: The ability to use tools to achieve goals.
- Memory: The ability to remember past interactions and use that information to inform future actions.

# Layered architecture for agentic systems
1. Perception Layer: Responsible for sensing and interpreting the environment.
2. Decision-Making Layer: Responsible for processing information and making decisions based on goals and perceptions.
3. Action Layer: Responsible for executing decisions and interacting with the environment.
4. Memory Layer: Responsible for storing and retrieving information from past interactions.
5. Tools/Action Layer: Responsible for utilizing tools and resources to achieve goals.

# ReAct Agent Loop 
1. Plan: The agent formulates a plan based on its goals and the current state of the environment.
2. Act: The agent executes the planned actions in the environment.
3. Observe: The agent observes the outcomes of its actions and gathers new information from the environment.
4. Reflect: The agent reflects on the outcomes and updates its knowledge and strategies based on the new information.
5. Repeat: The agent continues to iterate through the loop, refining its plans and actions over time to achieve its goals more effectively.
